## 차원의 저주

차원의 저주
- 고차원 공간에 있는 데이터를 분석할 때 발생하는 여러 가지 현상
- 훈련 샘플 각각이 수천 심지어 수백만 개 특성을 가져 많은 특성은 훈련을 느리게
  할 뿐만 아니라 좋은 솔루션을 찾기 어렵게 만듬
- 차원 = 변수의 수 = 축의 개수
- 차원이 늘어난다
  - 변수의 수가 많아진다
  - 축의 개수가 많아진다
  - 데이터의 공간이 커진다
  - 이를 채우기 위한 데이터 건수도 증가

고차원 공간은 직관적 상상 불가
- 3차원 큐브에서 임의의 두 점을 선택하면 평균 거리는 대략 0.66
- 만약 1000000차원의 초입방체에서 두 점을 무작위로 선택할 경우 평균 거리는
  약 428.25
- 훈련 세트의 차원이 클수록 과대적합 위험이 커짐

해결 방법
- 훈련 샘플의 밀도가 충분히 높아질 때까지 훈련 세트의 크기를 키우는 것
- 불행하게도 실제로는 일정 밀도에 도달하기 위해 필요한 샘플 수는 차원의 수가
  커짐에 따라 기하급수적으로 늘어남

차원 축소
- 변수 선택
  - 원래 있었던 변수 중 가장 효율적인 변수만 골라서 사용함
- 변수 추출
  - 원래 있었던 변수들을 새로운 효율적인 변수로 변형시켜 사용

PCA
- 정보의 손실을 최소화 하면서 차원을 감소시키는 방법
- 데이터들을 정사영 시켜 차원을 낮춘다면 어떤 벡터에 데이터들을 정사영 시켜야
  원래의 데이터 구조를 제일 잘 유지할 수 있을까?
- 기준이 되는 임의의 선에 수직으로 점을 이동시켜 값을 (x, y)가 아닌 a라는 새로운 값으로
   차원을 축소시킴 ( 2차원 데이터를 1차원 데이터로 변형시킴 )
- 정보를 잘 투영했다 판단하는 기준은 각 데이터가 임의의 선에 데이터를 수직으로 이동
  시켰을 때 얼마나 고르게 분포 되었는지로 판단함

정사영
- 특정 도형(또는 선분)에 빛을 쪼였을 때, 그 빛과 수직인 평면에 생기는 그림자


## 행렬과 행렬 연산

행렬의 합
- 행렬의 합은 그들이 같은 크기의 행렬일 때에만 정의됨
- 같은 크기의 행렬을 각각의 성분끼리 합하는 것

행렬의 곱 ( 시험 출제 가능성 매우 매우 높음 ) -> 시험 때 수식 다 적어야 됨( 계산 과정 )
- A=[a_ij]가 m * n 행렬이고 B = [b_ij]가 n*p 크기의 행렬인 경우
  행렬 A와 B의 행렬의 곱은 AB = C = [c_ij]인 m*p 행렬
- A와 B의 n이 같은 수 일 때만 곱이 가능함

공분산
- 평균: 주어진 수의 합을 측정 개수로 나눈 값
- 분산: 편차의 제곱의 평균값, 변량들이 퍼져있는 정도를 의미
- 편차: 변량(값) - 평균
- 표준편차: 분산은 수치가 너무 크기 때문에 제곱근으로 적당하게 줄여 사용하는 데
  이를 표준편차라 함
- 분산/표준편차의 의미
  - 표준편차가 크면, 수치들이 전반적으로 마구잡이 들쭉날쭉 하다는 의미
  - 표준편차가 작으면, 수치들이 대체로 비슷하고 고만고만 하다는 의미
- 공분산: 2개의 확률변수의 선형 관계를 나타내는 값
  - 특징 쌍들의 변동이 얼마나 닮았는가를 표현
  - 공분산 행렬은 일종의 행렬로, 데이터의 구조를 설명
  - [[3,2],[2,4]]라는 행렬이 있을 때 3은 x축 방향으로 빠진 정도(x축의 분산), 2는 (x,y)축 
   방향으로 함께 퍼진 정도, 1은 ( x, y ) 축 방향으로 함께 퍼진 정도, 4는 y축 방향으로
   퍼진 정도(y축의 분산), ( x, y )축 방향으로 함께 퍼진 정도는 항상
   같은 값임( 선분의 기울기 )
- 양수의 공분산: 하나의 값이 상승하는 경향을 보일 때 다른 값도 상승
- 음수의 공분산: 하나의 값이 상승하는 경향을 보일 때 다른 값이 하강

공분산의 의미
- 기하학적 표현: 행렬 선형 변환
- 하나의 벡터 공간을 선형적으로 다른 벡터 공간으로 mapping하는 기능 가짐
- 지금의 데이터의 분포를 원래의 원의 형태로 주어졌던 데이터가 선형 변환에 의해
  변환된 결과

공분산 행렬 ( 시험 문제 무조건 나옴 )
- 공분산 행렬에 따른 데이터 변환
- 2차원 이상의 변수들에 대해 공분산을 행렬로 표현한 것
- d: 데이터의 수, n: 차원의 수

행렬 X
- 각 열의 평균 값은 0인 행렬
- 중심축으로 움직이는 벡터
