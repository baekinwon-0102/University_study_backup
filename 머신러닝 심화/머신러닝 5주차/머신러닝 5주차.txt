## PCA

PCA 개념
- PCA(Principal Component Analysis)
  - 비지도 학습에서 자주 사용되는 차원 축소를 위한 주요 선형 알고리즘
  - 데이터 세트에서 유효한 근사값을 만드는 데 덜 유용한 기능을 식별하고 폐기
- PCA 활용 (5주차 pdf 5페이지 그림 시험 문제 가능성 높음)
  - 잡음이 있는 기능에 모델을 과대적합 경향 감소
  - 기계 학습 알고리즘 훈련 속도 향상
  - 효율적이고 간결한 데이터 시각화

공분산
- 특징 쌍들의 변동이 얼마나 닮았는가를 표현
- 공분산 행렬은 일종의 행렬로 데이터의 구조를 설명함

공분산의 의미
- 기하학적 표현: 행렬 선형 변환
- 하나의 벡터 공간을 선형적으로 다른 벡터 공간으로 mapping하는 기능 가짐
- 지금의 데이터 분포를 원래의 원의 형태로 주어졌던 데이터가 선형변환에 의해
  변환된 결과


## 고유 벡터와 고유 값

고유 벡터
- 행렬이 벡터에 작용하는 주축 표현
- 공분산의 고유 벡터는 데이터 분산을 표현

고유 벡터의 순서
- 고유 값이 큰 순서대로 중요도가 높음

고유 값
- 아래 수식을 만족하는 값
  A->x = t_1->x
- 선형 변환을 했을 때 방향이 변하지 않는 벡터의 크기

행렬 변환
- 벡터를 다른 형태로 변환
- 크기와 방향 모두 변환 됨

행렬식의 성질
- 행렬 A =[a,b  에 대해서
              c, d] 
  - 행렬의 열 중 하나에 k배 만큼 상수배 -> 행렬식의 값도 k배 커짐
  - 행렬의 열 중 하나가 0 벡터이면 행렬식의 값은 0
  - 행렬의 두 열이 동일하다면 행렬식의 값은 0
  - 행렬의 열 중 하나가 다른 열에 상수배라면 행렬식의 값은 0

역행렬
- 행렬 A = [a, b 에 대해서 A^-1 = 1/det(A)[ d -b
               c, d]                                   -c a ]
- 행렬식(daterminant)
  det(A) = ad-bc

역행렬의 성질
- 역행렬도 특별한 성질을 가지는 행렬이므로 역행렬 또한 선형변환으로 취급 가능
- 역행렬의 선형변환: 원래 행렬에 대한 역-선형변환
  - 원래 행렬의 선형변환: 넓이 1인 정사각형 -> 넓이 ad-bc 평행사변형
  - 역행렬의 선형 변환: 다시 한 변의 길이가 1인 정사각형으로 돌려줌
- AA^-1 = I = [1 0
                    0 1]

고유 벡터 변환에 대한 예시
- 변환 후에도 각도가 변하지 않는 것이 고유 벡터
- 변환 시 가장 긴 길이를 갖는 벡터가 높은 고유 값을 가지는 값
