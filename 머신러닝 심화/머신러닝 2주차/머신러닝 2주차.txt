
K-최근접 이웃 알고리즘 (k-Nearest Neighbor, kNN)
- 지도학습 내 분류 알고리즘
- 문제가 제시될 때 학습 데이터를 활용해 해답을 찾는 게으른 학습(lazy learning) 방법 ★
  ( 시험 출제 가능성 높음 )

최근접 이웃
- 이웃은 가장 가까이 존재하는 데이터
- 새로운 데이터가 입력되면 근접한 이웃 데이터 찾아서 이를 토대로 출력 결정
- 현재 데이터 속성값 기준으로 가까운 k개 데이터 찾아 k개의 레이블 중
  가장 많이 분류된 값으로 현재 데이터 분류
- ex) 지역구 분류

최근접 이웃의 특징 ( 시험 출제 가능성 매우 매우 높음 )
- 벡터 연산이 가능한 N차원 공간에서 활용 가능
- N개의 데이터 속성 포함 가능
- 위/경도와 같은 실제하는 N차원 속성이 아니어도 데이터 특성을 벡터 공간의 축으로
  지정해 데이터 간 거리 계산 가능
- 데이터 속성 활용해 차원 공간을 통한 거리 정의가 어려우면 적용 못 함 ★
   (수치화 불가능 시 적용 어려움)
- 분류시 동률 상황 피하기 위해 k 값 홀수로 지정하는 것이 유리함 ★

kNN 수학적 계산
- 벡터 공간 내 새로 입력된 데이터와 기존 학습 데이터 간 거리 계산
- 계산된 거리에서 가장 가까운 거리부터 먼 거리까지 지정된 k개 레이블 선정
- 이 중 가장 많이 집계된 레이블로 새로운 학습 데이터의 레이블 지정

N차원 공간에서의 두 지점 사이 기하학적 거리
- 벡터를 이용한 거리 계산
- 지점 A=(a1,a2,...an), 지점 B=(b1,b2,...bn)일 때 두 지점 사이 거리
  루트[(a1-b1)^2+(a2-b2)^2...(an-bn)^2] ( 시험 출제 가능성 매우 매우 높음 ) ★

kNN 장단점 ( 시험 출제 가능성 매우 높음 ) ★
- 장점
  1. 학습 알고리즘 직관적이고 이해 쉬움
  2. 숫자로 구분해 표기 가능한 속성을 지닌 데이터에 우수한 성능
  3. 별도의 모델 학습 과정 요구 없음
- 단점
  1. 미리 학습 안 하고, 데이터 입력 되었을 때 모든 학습 데이터 간 거리 계산 실시해서
     알고리즘 복잡성에 비해 예측 속도 느림 (계산 오래 걸림)
  2. 예측값이 지역 정보에 편향 되어 나타날 가능성 높음
