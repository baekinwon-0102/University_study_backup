디지털 신호와 아날로그 신호

- 디지털 신호는 구분 가능한, 제한된 신호 값을 가지고 불연속적으로 변하는 신호
- 아날로그 신호는 연속성을 가진 신호


- 편의 입출력 제어 -

LED
- 빛을 방출하는 다이오드
- 다이오드는 순방향 전류만 흐름
- 블랙베리의 LED는 캐소드가 아두이노의 Gnd에 연결 되어있음, 애노드는 D4에 연결되어 있음
- D4핀의 출력을 제어해 LED를 켜고 끌 수 있음

아두이노 LED 제어 과정
- 핀 번호 정의
- setup에서 LED 제어하는 핀을 출력으로 설정
  - pinMode 함수는 특정 핀을 입력 또는 출력으로 동작하도록 설정하는 함수임
  - 첫 번째 인자로는 설정할 핀 번호, 두 번째 인자로는 설정 값(INPUT, OUTPUT)을 넘김
- loop 에서는 RED LED를 각각 다른 타이밍으로 켜고 끄는 동작 반복
  - LED를 켜거나 끄기 위해 D4핀에서 HIGH또는 LOW를 출력하게 하려면 digitalWrite 함수를 사용해야 함

PWM
- PWM 핀으로 아날로그 신호와 비슷한 신호 만들 수 있다
- PWM은 Pulse Width Modulation의 약자로 디지털 펄스의 한 주기중에 신호가 High인 시간의 비율을 조정하는 변조 방식
- 펄스의 한 주기에서 High인 시간의 비율을 Duty Cycle이라 함
- 실제 신호는 디지털 펄스이지만 Duty Cycle에 비례하는 아날로그 전압을 출력하는 것 처럼 동작함

아두이노 LED 제어 과정 2
- 핀 번호 정의
- setup에서 LED 제어하는 핀을 출력으로 설정
- loop 에서는 RED LED를 각각 다른 타이밍으로 켜고 끄는 동작 반복
  - LED를 서서히 켜거나 끄기 위해 D4핀에서 0~5V 사이의 전압 출력하게 하려면 analogWrite 함수를 사용
  - 첫 번째 인자는 출력 설정할 핀 번호, 두번째는 출력 값 입력

스위치
- 블랙베리 후방엔 사용자를 위한 두 개의 스위치가 있고 이를 택트 스위치라 부름
- 스위치를 눌렀을 때만 리드 부분이 연결 됨

스위치 제어 과정
- RED LED와 스위치를 사용하기 위한 핀 번호 정의
- setup에서는 스위치의 상태를 출력하기 위한 시리얼 모니터 통신 설정함, RED LED 제어하는 핀은 출력으로, 스위치는 입력으로 함
- loop에서는 SW1과 2의 상태 값을 읽어 시리얼 모니터에 출력하고 SW1과 2의 상태 값이 서로 다를 때 RED LED를 켜는 동작 반복
- 핀의 전압 값을 읽기 위해 digitalRead 함수를 사용, 첫 번째 인자로 값을 읽을 핀 번호를 넘기면 함수는 해당 핀의 전압 값을 0 또는 1로 반환

부저 제어
- 부저 내부에는 얇은 판이 있음 피에조 효과에 의해 이 얇은 판에 전압을 가하면 얇은 판에는 기계적 변형이 생김
  또한 소리의 높낮이는 음파의 진동수에 따라 달라짐
- 이 점들을 이용해 특정 주파수에 전압 가하면 다른 소리가 나오게 할 수 있음

부저 제어 과정
- 부저의 핀 번호 정의
- 주파수와 박자의 길이를 밀리초로 정의
- 음악을 연주하는데 사용할 음계와 박자 배열 선언 후 초기화
- 이전에 연주된 음계를 저장하는 변수 선언
- tone(bin, freq): 특정 핀에 특정 주파수의 구형파를 발생시킴
- noTone(pin): 특정 핀에 tone 함수에 의해 시작된 구형파 발생 멈춤
- setup
  - 음계와 박자 배열의 길이가 다른지 확인, 배열의 길이가 다르면 진행 멈춤
  - 음계 배열의 길이만큼 for문 반복: 현재 연주하련느 음이 이전에 연주한 음과 같다면 각각 구분 위해 부저 소리 잠깐 멈춤, for문이 반복된 횟수에 따라 음계와 박자 배열에서 적절한 
   위치의 값을 연주, 현재 연주된 음을 prevNote 변수에 저장
- 연주 끝나면 부저 끔

RGB LED 제어
- RGB LED는 같은 LED 모듈을 여러 개 연결한 후 하나의 데이터 라인으로 모든 LED 제어 가능
- begin(): 데이터 핀을 OUTPUT으로 설정하고 통신 준비
- setBrightness(brightness): 밝기 지정
- Color(red,green,blue): 색상 설정
- setPixelcolor()
